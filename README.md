# README Insurance Tracker FE #

# General steps:
How to make the Frontend work?

    1. Install Dependencies.

    2. Create file `.env` based on `env.example`.
        - Remember to complete the file with the necessary information.

    3. Run application.

---
---

## 1 - Install Dependencies:

```bash
npm install
```

---

## 2 - Start FE server:
```bash
npm run dev
```
